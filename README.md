# mini-triangle-java

This is an implementation of a compiler for the Mini-Triangle programming language following the excellent textbook, "Programming Language Processors in Java" by Watt & Brown.

## Build

```
$ mvn clean && mvn compile
```

## Tests

```
$ mvn test
```
